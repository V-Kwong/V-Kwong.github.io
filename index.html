Notable Courses and Projects

CPEN 221 - Principles of Software Construction

Created an approximation of a relational database by implementing a multi-threaded server application, YelpDBServer. YelpDBServer wraps an instance of a database datatype, YelpDB, that parses JSON objects from an excerpt from the Yelp Academic Dataset and supports operations such as k-means clustering for grouping Yelp restaurants by their latitude and longitude location and least squares regression for predicting the rating that a user may give to a restaurant. YelpDBServer is able to concurrently handle adding and retrieving of objects in JSON format from requests made by more than one connection at a time.

CPEN 311 - Digital Systems Design

Created an RC4 decryption circuit by designing and programming a finite state machine that reads an encrypted message from a ROM and executes a ‘brute-force’ attack on the RC4 by cycling through the entire keyspace, stopping and writing the result to a RAM when a successful decryption is performed.

CPEN 321 - Software Engineering

Applied agile software development practices to develop LiveWire, an iOS application that works as a simple automated note taker. LiveWire uses iOS' Speech Recognition API and the Google Cloud Platform to authenticate users, record ideas and important notes on the fly, and store them as electronic transcripts.

CPEN 331 - Operating Systems

Implemented synchronization primitives and various file and process system calls such as close, write, dup2, __getcwd, and execv in OS/161, a simplified operating system that includes a kernel of conventional design and a simple userland, in C.

ELEC 291 - Electrical Engineering Design Studio I

Designed, assembled, and programmed an autonomous robot and a controllable guide wire transmitter. The robot follows the path of the guide wire by analyzing the magnetic field strength and by manipulating the magnetic field produced by the guide wire, the transmitter is able to send commands to the robot to operate forwards or backwards, turn right or left at the next intersection, start and stop at any point, or turn 180 degrees.
